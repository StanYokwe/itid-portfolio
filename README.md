# ITID Cartoon DVD Menu Portfolio

An original, interactive portfolio website inspired by early-2000s animated DVD menus. It uses playful motion, scene selection cards, bonus features, a remote-control navigation panel, and project chapter popups.

## Files

- `index.html` — website content and structure
- `styles.css` — early-2000s cartoon/DVD menu UI styling and animations
- `script.js` — interactivity, filters, project modals, random scene button, motion toggle

## How to publish on GitHub Pages

1. Create a public GitHub repository.
2. Upload these files directly into the repository root, not inside another folder.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Choose branch **main** and folder **/root**.
6. Save.

Your site will publish at either:

- `https://yourusername.github.io` if the repository is named `yourusername.github.io`
- `https://yourusername.github.io/repository-name` for a normal project repository

## Customize before publishing

Search inside `index.html` and replace:

- `Isaac Wani`
- `your.email@example.com`
- `https://github.com/yourusername`
- `https://www.linkedin.com/in/yourusername`

Add your real screenshots by replacing the CSS illustration blocks inside `.scene-screen` sections with image tags if desired.
