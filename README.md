# Wii-Style Portfolio Website

This is a GitHub Pages-ready static portfolio website for an IT Innovation & Design graduate. The layout is built as one large console-style channel menu, where each tile opens a project or portfolio section.

## Files

- `index.html` — website structure
- `styles.css` — console menu layout, animations, custom cursor styling
- `script.js` — interactive channel selection, modal content, clock, and cursor movement
- `assets/` — cursor image assets

## Customize

Open `script.js` and replace:

- `your.email@example.com`
- `https://github.com/your-username`
- placeholder project links

Open `index.html` and update:

- `Isaac Wani`
- the tagline if needed

Add your resume PDF to `assets/`, then update the resume link in `script.js`.

## Publish on GitHub Pages

1. Create a new public GitHub repository.
2. Upload the contents of this folder, not the folder itself.
3. Make sure `index.html` is in the root of the repository.
4. Go to repository Settings > Pages.
5. Deploy from the `main` branch and root folder.

## Notes

The hand cursor uses the image you uploaded. If you want this portfolio to be fully original for professional/commercial use, consider replacing the cursor with your own hand-pointer illustration.
